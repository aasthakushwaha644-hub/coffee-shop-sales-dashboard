# Coffee Shop Sales Dashboard ☕

This project presents an interactive Coffee Shop Sales Dashboard built using Power BI to analyze sales performance, revenue trends, and product insights.

## 📊 Project Overview
The dashboard provides insights into:
- Total Revenue
- Total Orders
- Total Quantity Sold
- Sales by Category
- Top Selling Products
- Monthly Sales Trend
- Store Location Performance

## 🛠 Tools Used
- Microsoft Power BI
- Data Visualization
- Data Analysis

## 📈 Key Features
- Interactive filters
- Dynamic charts
- KPI cards
- Category wise sales analysis
- Monthly revenue trends
